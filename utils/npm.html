npm基本使用 #

登录 #

bash
npm login

查看登录信息 #

bash
npm who am i

发布包 #

bash
npm publish

❗ 发布包到npm时，需设置成npm源， 不然会报403错误，提示找不到该用户

删除已发布包 #

bash
npm unpublish [<pkg>][@<version>] --force

把本地项目软链接到全局 #

在项目目录执行

bash
npm link

或者

bash
npm install  [<pkg>][@<version>] -g

删除全局软链接 #

bash
npm unlink  [<pkg>][@<version>] -g

或者

bash
npm uninstall  [<pkg>][@<version>] -g

❗ 由于npm会把当前项目的package.json的name作为项目名称软链接到当前node版本的node_modules文件夹下，所以删除时要使用本地项目package.json的name，而不是项目目录名
